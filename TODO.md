## Фаза 5: Эксперимент 2 — Наблюдения и предсказательная сила (2–3 недели)

**Гипотеза:** когерентные ошибки дали ΔLoss ≈ 0, потому что задачи самодостаточны. Добавление наблюдений восстановит truth bias.
**Внимание:** расхождения могут быть регулярными и сжимаемыми (a×b=a×(b-1) всегда занижает на a). Эффект может быть слабым.

- [ ] Обучить когерентные 50/50 **с наблюдениями** (условие B), 4 seeds
- [ ] Варьировать долю наблюдений: 0%, 10%, 25%, 50%, 100%

## Фаза 6: Эксперимент 3 — Ad hoc escape hatches (2–3 недели)

**Гипотеза:** ad hoc объяснения (аналог гомеопатии) несжимаемы и дадут сильнейший truth bias.

Пять условий для ложной теории:

| Условие | Описание | Аналогия | Ожидаемый ΔLoss |
|---------|----------|----------|-----------------|
| A | Без наблюдений | Теория в учебнике | ≈ 0 (проверено) |
| B | Голые расхождения | Фальсифицированная теория | Средний |
| C | Ad hoc escape hatches | Гомеопатия | Сильный |
| D | Систематическая коррекция | Ложная теория с одним фиксом | Слабый (≈ 0?) |
| E | Неконкретные предсказания | Астрология | Слабый–средний |

**Ожидаемый порядок:** ΔLoss(C) > ΔLoss(B) > ΔLoss(E) > ΔLoss(D) ≈ ΔLoss(A) ≈ 0

- [ ] Реализовать генератор ad hoc объяснений (условие C):
  - Пул шаблонов с рандомизированными деталями, каждое объяснение уникально
  - Формат: `Prediction: 40. Observation: 50. Explanation: [уникальная причина]. Corrected: 50 ✓`
- [ ] Реализовать систематическую коррекцию (условие D):
  - Одно правило для всех: «always add first operand to result»
- [ ] Реализовать неконкретные предсказания (условие E):
  - «result is moderate» без точных чисел
- [ ] Обучить все 5 условий, по 4 seeds каждое (20 моделей)
- [ ] Проверить порядок ΔLoss
